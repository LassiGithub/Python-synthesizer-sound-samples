Introduction

These are sound samples for my python synthesizer project. The structure of this page roughly follow the jupyter notebook file of that original project, so you should be able to quite easily find the corresponding sound examples. If you're here from the machine learning project, sound samples of the model's predictions are also below.

Machine Learning Predictions with Randomized Sounds

These examples contain more sounds with randomized parameters, and also predictions made by a machine learning model. A sound with random parameters is first created, then the sound is given to a model, that will predict the parameters. Each of these example pairs contains first a sound synthesized with the original parameters, then a sound synthesized with the predicted parameters. In the synthesizer notebook file, this would be the "generated sound prediction" section.

As you can hear, the predictions aren't perfect, but they are quite close! It's almost like having a different version of the same sound.

Machine Learning Predictions with Recorded Sounds

Similar to above, we have predictions made by the machine learning model. However, this time it's predicting parameters for sounds that aren't from a synthesizer. I recorded these sounds with a microphone, and most of them are impossible to recreate with this synthesizer. It's interesting to see how the model will attempt to do that. In the synthesizer notebook file, this would be the "non-synth sound prediction" section.

Perhaps as expected, the model is having quite a difficult time with these. It's clear that what the model learned was not the overall idea of the sounds in the same way as a human understands them. It must have picked up on some more specific patterns, that it tries to find in these recorded sounds.
